About the Project

This project was inspired by my experience working in the HR department of a small company. I saw firsthand how challenging it can be to manage employee data and track attendance using outdated systems. This inspired me to create a more efficient and user-friendly HR management system.

Timeline: This project was developed as part of my portfolio for Holberton School.
